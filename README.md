
# PV-Ertrags-Simulator Baden-Wuerttemberg (Konstanz & weitere Staedte)

Dieses Projekt enthaelt eine einfache Streamlit-App, mit der Sie fuer
PV-Anlagen zwischen 10 kWp und 50 MWp in Konstanz und ausgewaehlten
Staedten in Baden-Wuerttemberg Jahresertraege, Eigenverbrauch, Ueberschuss
sowie Speicherwirkung in drei Szenarien (konservativ, realistisch,
optimistisch) abschaetzen koennen.

Installationsschritte:

1. Python 3.10+ installieren.
2. Projekt entpacken und in das Verzeichnis wechseln.
3. Pakete installieren:

   pip install -r requirements.txt

4. App starten:

   streamlit run app.py

Die verwendeten Ertragswerte und die Speicherlogik sind bewusst vereinfacht
und dienen der Grobplanung.
